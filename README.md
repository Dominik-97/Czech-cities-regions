# Czech-cities-regions

This repository contains list of regions of Czech republic with all available postal codes in xlsx, csv, xml and json formats.

Data description:

| Column | Description |
| ------ | ----------- |
| String | PSČ + Město |
| PSČ    | PSČ         |
| CC     | Count       |
| Město  | Město       |
| Okres  | Okres       |
| Kraj   | Kraj        |
